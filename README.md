## 📖 About

**Goalz** is a modern web application built to help users set, track, and achieve their personal and professional goals. It is designed for individuals who want to stay organized and motivated while working towards their objectives. The app focuses on simplicity, speed, and an intuitive user experience.

---

## ✨ Features

• ✅ Create and manage goals with custom categories
• ✅ Track progress with visual indicators and milestones
• ✅ Set deadlines and reminders for accountability
• 🔜 Upcoming: Collaborative goals with team members
• 🔜 Upcoming: Mobile app integration
• 🔜 Upcoming: AI-powered goal suggestions

---

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, TailwindCSS
- **Framework**: Next.js
- **Styling**: TailwindCSS
- **State Management**: React Hooks / Context API
- **Testing**: Jest, React Testing Library

## 🚀 Getting Started

### Prerequisites

```bash
node >= 18.x
npm >= 9.x (or yarn)
git
```

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/sohamchavan07/Goalz.git
cd Goalz

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env.local

# 4. Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
